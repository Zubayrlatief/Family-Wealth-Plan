# Family Wealth Plan

A private, single-file family finance dashboard: household cash flow, bond payoff scenarios, retirement income projection, and investable-asset growth to 2035.

- **`index.html`** — the whole app. No build step, no dependencies. Open it in a browser or deploy as a static site (Vercel/GitHub Pages).
- Three scenarios (Drift / Discipline / Builder) with adjustable assumptions: bond balance, bond rate, GEPF pension estimate, side-income target.
- Charts are hand-rolled SVG with hover tooltips; checklist state persists in `localStorage`; light and dark themes.

All figures are an educational planning model, not financial advice.
